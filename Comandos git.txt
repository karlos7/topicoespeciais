git --version (Verifica a versão do git)
git init (Inicia um repositório vazio na pasta atual)
git config --global user.name "Karlos Oliveira" (Cria usuário do git)
git config --global user.email "karlos.1914.so@gmail.com" (Cria email do git)
git status (Verifica tudo que não foi salvo)
git add . (Coloca os arquivos na área de transferência)
git commit -m "Mensagem" (Informa uma mensagem para identificar o cmmmit)
git log (Mostra todo o log dos commits realizados)
git remote add origin linkdorepositorio (Vincula a pasta local ai repositório do git)
git push -u origin master (Manda os arquivos para a nuvem)
